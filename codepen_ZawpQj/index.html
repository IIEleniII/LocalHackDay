IEEE AUTh SB : MLH Local Hack Day


Local Hack Day Info

To MLH Local Hack Day είναι μία ημέρα αφιερωμένη στην δημιουργία τεχνολογίας με σκοπό να φέρει σε επαφή και να ενισχύσει την συνεργασία προγραμματιστών (και όχι μόνο) σε κάθε φοιτητικό campus! Το Σάββατο 2 Δεκεμβρίου, φοιτητές σε περισσότερα από 300 μέρη σε όλο τον κόσμο θα λάβουν ταυτόχρονα μέρος σε αυτή την γιορτή της τεχνολογίας!

Μέσα σε 12 ώρες οι συμμετέχοντες θα έχουν την ευκαιρία να παρακολουθήσουν 3 workshops που δημιουργήθηκαν από το Github ειδικά για την ημέρα αυτή, να συνεργαστούν σε ομάδες, να υλοποιήσουν και να παρουσιάσουν μία ιδέα, αλλά και να λάβουν μέρος σε διάφορα fun και social activities!
